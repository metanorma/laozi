
== 第五章

[#L5-1]
天地不仁，以萬物為芻狗。

[#L5-2]
聖人不仁，以百姓為芻狗。

[#L5-3]
天地之間，其猶橐籥與？

[#L5-4]
虛而不屈，動而愈出。

[#L5-5]
多言數窮，不如守中。
