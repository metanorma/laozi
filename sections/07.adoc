== 第七章

[#L7-1]
天長地久。

[#L7-2]
天地所以能長且久者，以其不自生也，故能長生。

[#L7-3]
是以聖人後其身而身先，外其身而身存。

[#L7-4]
不以其無私與，故能成其私。
