
== 第十三章

[#L13-1]
寵辱若驚，貴大患若身。

[#L13-2]
何謂寵辱若驚？寵為下，得之若驚，失之若驚，是謂寵辱若驚。

[#L13-3]
何謂貴大患若身？吾所以有大患者，為吾有身；及吾無身，吾有何患？

[#L13-4]
故貴以身為天下，若可寄天下。愛以身為天下，若可託天下。
