== 第十一章

[#L11-1]
三十輻共一轂，當其無，有車之用。

[#L11-2]
埏埴以為器，當其無，有器之用。

[#L11-3]
鑿戶牖以為室，當其無，有室之用。

[#L11-4]
故有之以為利，無之以為用。
